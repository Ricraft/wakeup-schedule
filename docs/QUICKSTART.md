# 快速开始指南

## 5分钟上手 WakeUp 课表

### 第一步：启动应用程序

```bash
cd wakeup-schedule
python main.py
```

### 第二步：设置学期信息

1. 点击菜单 **设置 > 学期设置**
2. 设置学期开始日期（通常是第一周的周一）
3. 查看当前周次是否正确
4. 点击"确定"保存

### 第三步：添加第一门课程

1. 点击工具栏的 **添加课程** 按钮
2. 填写课程信息：
   - **课程名称**: 例如"高等数学"（必填）
   - **教师**: 例如"张老师"
   - **地点**: 例如"教学楼A101"
   - **星期**: 选择上课的星期几
   - **节次**: 选择开始和结束节次（例如第1节至第2节）
   - **周次**: 选择上课的周次范围（例如第1周至第16周）
   - **单双周**: 选择每周/单周/双周
   - **备注**: 可选的备注信息
3. 点击"保存"

### 第四步：查看课表

- 课程会自动显示在课表中
- 使用不同的颜色区分不同课程
- 点击课程单元格可以查看详情、编辑或删除

### 第五步：切换周次

- 使用工具栏的 **上一周** / **下一周** 按钮
- 或在周次选择器中直接输入周次
- 点击 **回到当前周** 快速返回

## 常见操作

### 编辑课程

1. 点击课表中的课程单元格
2. 在弹出的对话框中点击"编辑"
3. 修改信息后点击"保存"

### 删除课程

1. 点击课表中的课程单元格
2. 在弹出的对话框中点击"删除"
3. 确认删除

### 自定义时间段

1. 点击菜单 **设置 > 时间段设置**
2. 修改每节课的开始和结束时间
3. 可以添加或删除节次
4. 点击"确定"保存

### 导入课表（从文件）

#### 从 HTML 导入
1. 从教务系统导出课表为 HTML 文件
2. 点击菜单 **导入 > 从HTML导入**
3. 选择 HTML 文件
4. 确认导入

#### 从 Excel 导入
1. 准备 Excel 课表文件（第一列=周一，第一行=第一节）
2. 点击菜单 **导入 > 从Excel导入**
3. 选择 Excel 文件
4. 确认导入

#### 从文本导入
1. 准备文本文件，每行一门课程
2. 格式：`周一 1-2节 高等数学 张三 A101 1-16周`
3. 点击菜单 **导入 > 从文本导入**
4. 选择文本文件
5. 确认导入

## 提示和技巧

### 💡 颜色管理
- 同名课程自动使用相同颜色
- 颜色基于课程名称自动分配
- 15种预设颜色，高对比度易识别

### 💡 冲突检测
- 添加或编辑课程时自动检测时间冲突
- 如果有冲突，会显示冲突的课程列表
- 可以选择继续保存或取消

### 💡 单双周课程
- 支持每周、单周、双周三种类型
- 单周课程只在奇数周显示
- 双周课程只在偶数周显示

### 💡 数据安全
- 程序关闭时自动保存数据
- 每次保存前自动创建备份文件
- 数据存储在 `%APPDATA%\WakeupSchedule\`

### 💡 快捷键
- `Ctrl+N`: 新建课表
- `Ctrl+S`: 保存课表
- `Ctrl+A`: 添加课程
- `Ctrl+Q`: 退出程序
- `Delete`: 删除课程

## 故障排除

### 问题：程序无法启动
**解决方案**:
1. 确保已安装 Python 3.10+
2. 确保已安装所有依赖：`pip install -r requirements.txt`
3. 检查是否有错误信息

### 问题：课表不显示
**解决方案**:
1. 检查是否已添加课程
2. 检查当前周次是否在课程的周次范围内
3. 检查单双周设置是否正确

### 问题：数据丢失
**解决方案**:
1. 检查 `%APPDATA%\WakeupSchedule\` 目录
2. 查找 `.bak` 备份文件
3. 将备份文件重命名为 `schedule.json`

### 问题：时间显示不正确
**解决方案**:
1. 打开 **设置 > 学期设置**
2. 检查学期开始日期是否正确
3. 重新设置学期开始日期
